# TurfConnect Design System: Athletic Synergy

## Overview
TurfConnect is a professional turf booking platform designed to connect athletes with high-quality sports facilities. The design language, "Athletic Synergy," balances energetic sporting vibes with clean, functional utility.

## Brand Identity
- **Name:** TurfConnect
- **Personality:** Professional, Energetic, Reliable, Tech-forward.
- **Tone:** Encouraging and focused.

## Color Palette
The palette is rooted in deep greens to evoke the feeling of a well-maintained pitch, supported by a clean slate-and-white foundation.

| Role | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary** | `#1B5E20` | Brand identity, primary buttons, active states. |
| **Secondary** | `#4CAF50` | Highlights, badges, "Instant" booking status. |
| **Accent** | `#FF6D00` | CTA buttons (View Details, Book Now), ratings. |
| **Surface** | `#F4FAFF` | Main application background. |
| **Surface Dim** | `#CFDCE4` | Sidebars and secondary containers. |
| **On Surface** | `#1A1C1E` | Primary text and headings. |
| **Error** | `#B00020` | Destructive actions and alerts. |

## Typography
- **Primary Font:** Inter (Sans-serif)
- **Headings:** Bold weights (700+) for clear hierarchy.
- **Body:** Regular and Medium weights (400-500) for readability.
- **Scales:**
  - `headline-lg`: 32px / 40px line-height
  - `headline-md`: 24px / 32px line-height
  - `body-md`: 16px / 24px line-height
  - `label-sm`: 12px / 16px line-height

## UI Components
### 1. Navigation
- **Sidebar:** Fixed left-side navigation for Desktop. Features a brand logo, primary navigation links (Home, Book Turf, History, Statistics), and a secondary footer section (Help, Logout).
- **Top Bar:** Contextual actions, search bars, and user profile management.

### 2. Cards & Containers
- **Venue Cards:** Grid-based cards featuring high-quality imagery, ratings, pricing, and specific amenity tags (e.g., Showers, Parking).
- **Stat Cards:** Dashboard-level summaries with icons and percentage growth indicators.

### 3. Interactive Elements
- **Buttons:**
  - *Primary:* Solid `#1B5E20` with white text.
  - *Action:* Solid `#FF6D00` for high-conversion steps like "Book Now" or "View Details".
  - *Outlined:* Bordered variants for secondary actions like "Details" or "Cancel".
- **Inputs:** Clean, outlined fields with clear icons and labels.

### 4. Layout
- **Grid:** 12-column responsive grid for desktop.
- **Spacing:** Based on an 8px base unit (Margins: 24px/32px, Padding: 16px/24px).
- **Rounding:** `ROUND_EIGHT` (8px border-radius) applied to buttons, cards, and input fields.

## Design Principles
1. **Focus on Action:** Key workflows (Searching and Booking) should be prominent and require minimal clicks.
2. **Visual Social Proof:** Integration of ratings and player feedback to build trust.
3. **High Contrast:** Ensure all interactive elements are clearly distinguishable from background surfaces.
