# Project Brief: TurfConnect

## 1. Project Overview
**TurfConnect** is a professional, high-fidelity web platform designed to streamline the discovery and booking of sports facilities. Built with a focus on ease of use for athletes and efficient management for venue owners, the platform bridges the gap between available "pitches" and the teams that play on them.

## 2. Brand Identity: "Athletic Synergy"
The platform's visual language is rooted in the energy of professional sports balanced with modern utility.
*   **Tone:** Professional, Energetic, Reliable, Tech-forward.
*   **Color Palette:** 
    *   **Primary Green (#1B5E20):** Represents the pitch and reliability.
    *   **Accent Orange (#FF6D00):** Drives high-conversion actions (Booking, Details).
    *   **Surface Slate (#F4FAFF):** Ensures a clean, readable data environment.
*   **Typography:** **Inter** (Sans-serif) for high legibility and a contemporary feel.

## 3. Core Functional Requirements
### A. User Authentication
*   Secure login and signup flow for athletes.
*   Social login integration (e.g., Google) for reduced friction.

### B. Venue Discovery (The "Explore" Workflow)
*   **Search & Filter:** Find venues by location, sport type (Soccer, Cricket, Basketball, etc.), price range, and amenities (Parking, Showers, Floodlights).
*   **Visual Browsing:** High-quality imagery with clear status badges (e.g., "Trending," "Pro Rated").

### C. Booking Engine
*   **Real-time Availability:** Visual calendar and time-slot selection.
*   **Dynamic Pricing:** Instant calculation of totals based on duration and hourly rates.
*   **Seamless Checkout:** Clear path from slot selection to booking confirmation.

### D. User Dashboard
*   **Personalized Metrics:** Tracking total bookings, reward points, and favorite venues.
*   **Match Schedule:** Immediate visibility into upcoming games and navigation directions.
*   **Re-engagement:** Recommended turfs based on user behavior.

## 4. Design & UX Principles
1.  **Action-First Architecture:** Minimize the clicks required to find and book a slot.
2.  **Social Proof:** Integrated ratings and community reviews to build trust in venue quality.
3.  **High Contrast UI:** Clear distinction between background surfaces and interactive CTA elements.
4.  **Responsive Layout:** Desktop-first design with a persistent sidebar for efficient navigation.

## 5. Technical Specifications (Current Scope)
*   **Frontend:** React/Tailwind CSS based on the established design system tokens.
*   **State Management:** Real-time updates for slot availability.
*   **Device Support:** Desktop-optimized (1440px width).

## 6. Success Metrics
*   **Conversion Rate:** Percentage of users who complete a booking after visiting a venue page.
*   **User Retention:** Frequency of return bookings via the personalized dashboard.
*   **Booking Efficiency:** Time taken from landing to successful slot reservation.
